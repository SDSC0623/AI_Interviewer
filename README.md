# 智能多模态面试评测系统

基于AI技术的智能面试评测系统，集成多模态数据分析、实时语音识别、情绪分析等技术，为求职者提供专业面试训练服务。

## 🚀 核心特性

### 多模态智能评测
- **文本分析**：NLP技术分析回答的专业性、逻辑性
- **语音分析**：实时语音识别与情感分析
- **视觉分析**：面部表情、头部姿态、眼神交流评估
- **综合评分**：多维度数据融合生成客观评分

### 智能对话系统
- 大模型驱动的AI面试官
- 动态生成个性化面试问题
- 专业领域知识库支持

### 实时反馈报告
- 可视化能力雷达图
- 详细改进建议
- 历史趋势分析
- 专业HTML报告

## 🛠️ 技术架构

### 技术栈
- **前端**：tkinter/ttkbootstrap GUI + Flask Web界面
- **后端**：Python + Flask + 星火大模型 + OpenCV + TensorFlow
- **存储**：JSON文件 + 配置管理

### 核心模块
- **多模态评测**：6大能力维度评估（专业知识25%、技能匹配20%、语言表达20%、逻辑思维15%、创新能力10%、适应能力10%）
- **智能反馈**：可视化图表、雷达图、趋势分析
- **情绪分析**：实时表情识别、姿态分析、眼神评估
- **语音处理**：实时识别、情感分析、流畅度评估

## 📊 评测维度

- **专业能力**：理论基础、实践经验、技术深度、知识广度
- **软技能**：沟通表达、逻辑思维、情绪管理、适应能力
- **非语言行为**：面部表情、眼神交流、头部姿态

## 🎯 应用场景

- **个人求职者**：面试训练、简历优化、表现追踪
- **企业HR**：候选人筛选、面试标准化、效率提升
- **教育培训**：职业培训、就业指导、能力认证

## 🔧 快速开始

### 环境要求
Python 3.8+、Windows 10/11、摄像头麦克风

### 安装运行
```bash
# 安装依赖
pip install -r requirements.txt

# 配置API（编辑config.yaml）
spark_ai:
  APP_ID: "your_app_id"
  API_KEY: "your_api_key"
  API_SECRET: "your_api_secret"
(目前用的是我们自己的api，可以从设置里修改)

# 启动应用
python main.py        
```

### 使用流程
注册登录 → 上传简历 → 选择岗位 → 开始面试 → 获取报告
